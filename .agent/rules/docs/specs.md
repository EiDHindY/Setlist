# 🍱 App Specifications & Features: Setlist

## 🌟 The "Clash" (Core MVP Feature)

A **Clash** is a competitive musical event created by a host. It involves multiple users drafting songs and battling them out in real-time.

### 1. Clash Configuration (The Setup)
*   **Host-Driven:** Only the host can start and manage the clash state.
*   **Parameters:** 
    *   **Song Count:** 8, 16, or 32 songs.
    *   **Draw Style:** World Cup (Bracket) or Random Face-off.
    *   **Scoring:** Simple Voting (initial mode).
    *   **User Count:** Flexible, based on host invitation.

### 2. The Lobby & Invitation
*   Host invites users directly.
*   "Request to Join" system for non-invited users (Host approves/declines).
*   Host assigns the "Draft Quota" (how many songs each user adds to the total).

### 3. The Draft Phase
*   Users search YouTube/Library to add their assigned number of songs.
*   Clash moves to "Draw Phase" once the total song count is reached.

### 4. The Battle & Listen Party
*   **Synced Room:** Ability for all users to join a "Listen Party" where the YouTube audio/video plays in perfect sync for everyone.
*   **Voting:** Users vote on the current 1v1 matchup.
*   **Winner:** Moves forward in the bracket or stays on the leaderboard.

### 5. Persistence & Replay
*   Every Clash is saved with its full history (songs, users, match winners, and votes).
*   Ability to "Replay" a clash to see the progression.
