# ⚙️ Tech Stack & Dependencies: Setlist

## 🛠️ Core Technologies

* **Backend:** ASP.NET Core 10 (C#) Web API + SignalR.
* **Web Frontend:** Next.js 16 (React 19) + Tailwind CSS v4.
* **Mobile App:** Flutter (Dart).
* **Database:** Supabase (PostgreSQL).
* **Cache & Real-time State:** Upstash (Redis).
* **Authentication:** Supabase Auth.

## 📦 Distribution / Hosting Plan (The Budget Path)

* **Phase 1 (Development):**
  * Backend: Render.com (Free Tier)
  * Web: Cloudflare Pages (Free Tier)

* **Phase 2+ (Production):**
  * Backend: Hetzner or Contabo VPS (~$5/mo)
  * Web: Cloudflare Pages
* **Mobile:** Direct APK downloads (Android) or App Store/Play Store (Production).

## 🔌 External Services/APIs

* **Music Data:** YouTube Data API v3 (For searching songs and getting audio/video data).
